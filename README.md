# Amazon Financial Analysis & Forecasting Model

Financial model including forecasting, cost analysis, and DCF valuation.

## Overview
This project focuses on Amazon’s financial performance through forecasting and valuation analysis.

## Key Components
- Revenue forecasting
- Cost analysis
- Profitability analysis
- DCF valuation
- Financial assumptions and investment analysis
